DIGITAL FABRICATION
AND
PROTOTYPING FUNDAMENTALS

Digital Fabrication & Prototyping Fundamentals (EP1000) is an elective module offered at Singapore Polytechnic. In this module you will learn how to use Digital Fabrication techniques to design and create prototypes. Things like Computer Aided Designing techniques, 3D printing, lazer cutting and work documentation will be taught.

This website contains the documentation of assignments and projects that I would go through in my this elective module, Digital Fabrication and Prototyping. It also contains some information about me and my thought processes throughout this module. Nevertheless, I would want to enjoy myself in this module.
